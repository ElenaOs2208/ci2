A07 – POV-Ray rendering of seven acetic acid molecules
Repository structure A07/
│── acoh.smi               # SMILES structure of acetic acid
│── molecule.pov           # POV-Ray description generated by Open Babel
│── babel_povray3.inc      # Open Babel POV-Ray include file
│── acoh7.pov              # Final POV-Ray scene (seven molecules)
│── acoh7.png              # Rendered output image
│── README.md
1. Clone the repository
git clone https://github.com/ElenaOs2208/ci2.git
cd ci2/A07

cd REPOSITORY_NAME/A07
2. Install required software (Ubuntu)
Install Open Babel:
sudo apt install openbabel
Install POV-Ray:
sudo apt install povray
3. Convert the SMILES file to a POV-Ray file (Run inside the A07 folder:)
obabel acoh.smi -O molecule.pov --gen3D
4. Render the final image using POV-Ray
povray +Iacoh7.pov +Oacoh7.png +W1920 +H1080 +A
